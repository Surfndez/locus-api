## Add-on icon

How to use it:

 - extract content of *ic_launcher.zip* file
 - copy content of the 'ic_launcher/res' directory into own application *res* folder
 - modify all icons to match your own needs (you may contact if help is needed)
 - set icon for you application in manifest.xml as *android:icon="@mipmap/ic_launcher_no1*
 
Icon *ic_launcher_512.png* should be modified for the Google Play listing.
